Setting Up Flask Framework

Installation

First I created a folder for my project and opened it in Visual Studio Code.

In the terminal I setup a virtual environment and installed Flask. The I command used was:

pip install flask

Page Creation

The webpage was created in an HTML template file and rendered by the Flask application using the render_template function.

Challenges

I did not encounter any issues with setup and using Flask.

However, when I tried to host my app using Vercel, I encountered some issues with deploying my site.

I was able to find my error and successfuly deploy my app by following online tutorials.
